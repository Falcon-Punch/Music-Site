# Music-Site

Project to clone a music streaming style website similar to Spotify.<br/>
This project has helped me get familiar with a variety of languages and technologies including:<br/>
* PHP
* Advanced CSS
* JavaScript and jQuery
* Ajax
* MySQL
* Git Command Line 
* Sublime Text Editor
* Xampp
